# Santehnika Website Project

Это статический сайт интернет-магазина сантехники. На сайте представлены товары, категории продукции, страницы товаров и интерфейс с изображениями и визуальными блоками.

## Как запустить

Открой файл `index.html` в браузере.
Проект не требует установки дополнительных программ или серверов.
